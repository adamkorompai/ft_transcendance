Welcome to Transcendance!

A modern version of the classic video game Pong which simulates table tennis.

How it works?

Two players control paddles on either side of the screen, using them to hit a ball back and forth. The goal is to make the ball pass your opponent's paddle to score points, while preventing the ball from passing your own paddle.

The game is simple, with the ball moving increasingly faster as the rally continues, making each successive return more challenging. The first player to reach a set score wins the game.
